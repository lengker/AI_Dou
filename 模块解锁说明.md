# 模块解锁说明

本文档用于说明当前版本中，各场景、玩法与功能模块的实际解锁方式。内容以现有代码逻辑为准，便于策划、前端和测试统一对照。

## 1. 初始状态

- 完成映射后进入房间场景。
- 初始只解锁 `办公区`。
- 初始功能状态：
  - 未解锁 `生活区`
  - 未解锁 `户外区`
  - 未解锁 `图鉴 / 收藏柜`
  - 未解锁 `娱乐模块 / 推币机`
  - 未解锁 `抓娃娃机`

## 2. 主线解锁顺序

当前主线步骤为：

`intro -> bedroom -> forest -> atlas -> arcade -> claw -> free`

对应含义如下：

| 主线阶段 | 含义 |
|---|---|
| `intro` | 初始阶段，仅办公区可用 |
| `bedroom` | 已解锁生活区 |
| `forest` | 已解锁户外区 |
| `atlas` | 已解锁图鉴 / 收藏柜 |
| `arcade` | 已解锁娱乐模块 / 推币机 |
| `claw` | 已解锁抓娃娃机 |
| `free` | 主线完成，进入自由培养阶段 |

## 3. 场景解锁方式

### 3.1 生活区

解锁条件：

- 在 `办公区` 首次点击任意有效办公热区后解锁。
- 实际会在第一次有效办公区探索时，把主线从 `intro` 推进到 `bedroom`。

结果：

- 开放底部 `生活区` Tab。
- 弹出“卧室已接通”主线对话。

### 3.2 户外区

解锁条件：

- 在 `生活区` 中，首次使用以下任一恢复类互动时解锁：
  - `床`
  - `冰箱`

说明：

- 满足条件后，主线从 `bedroom` 推进到 `forest`。

结果：

- 开放底部 `户外区` Tab。
- 弹出“数据林海开放”主线对话。

## 4. 功能模块解锁方式

### 4.1 图鉴 / 收藏柜

解锁条件：

- 首次获得任意藏品时自动解锁。

当前可触发来源包括：

- 翻垃圾桶获得藏品
- 离线结算获得藏品
- 彩蛋获得藏品

补充说明：

- 在户外区成功记录动物时，如果此前还未解锁图鉴，也会一并补开图鉴主线。

结果：

- 右下角文件夹按钮正式可用。
- 主线从 `forest` 推进到 `atlas`。
- 弹出“图鉴开始记录”主线对话。

### 4.2 娱乐模块 / 推币机

解锁条件：

- 在 `户外区` 成功解锁并记录 `3 个动物点位` 后自动解锁。

说明：

- 这里统计的是 `forestAnimals.length >= 3`。
- 解锁时会同时补上：
  - `arcade`
  - `coin`

结果：

- `生活区街机` 与 `户外区对应街机入口` 可进入娱乐模块。
- 主线从 `forest` 或 `atlas` 推进到 `arcade`。
- 弹出“娱乐模块上线”主线对话。

### 4.3 抓娃娃机

解锁条件：

- 进入娱乐模块后，先完成 `一局推币机`。
- 结算时调用 `recordArcadePlay('coin')`，若此前未解锁抓娃娃机，则自动开放。

结果：

- 在小游戏大厅中解锁 `抓娃娃机` 卡片。
- 主线从 `arcade` 推进到 `claw`。
- 弹出“抓娃娃机开放”主线对话。

### 4.4 自由培养阶段

解锁条件：

- 抓娃娃机开放后，再完成 `一局抓娃娃机`。

结果：

- 主线从 `claw` 推进到 `free`。
- 弹出“世界开始完整”主线对话。

## 5. 场景内具体入口说明

### 5.1 办公区

- 电脑、桌面、书架、公告板等有效热区都可推进初始主线。
- 首次有效探索后会解锁 `生活区`。

### 5.2 生活区

- `床` 或 `冰箱`：在主线处于 `bedroom` 时，首次使用会解锁 `户外区`。
- `街机`：只有在 `arcade` 功能已解锁后才可进入小游戏大厅。
- 若 `arcade` 未解锁，点击街机会提示：
  - `娱乐模块还在校准中，先继续完善分身的生活环境吧。`

### 5.3 户外区

- 各动物交互点：用于推进林海记录。
- 当成功记录动物总数达到 3 个时，自动解锁娱乐模块。
- 户外区对应的街机入口也受 `arcade` 解锁状态控制。

## 6. 收藏与家具相关说明

### 6.1 收藏柜

- 只有在 `collection` 解锁后，右下角收藏柜按钮才会完全可用。
- 未解锁时点击会提示：
  - `先完成一次有效收集，图鉴才会开始记录。`

### 6.2 家具

- 家具不属于主线模块解锁。
- 家具通过点击对应热区后，消耗碎片单独解锁。
- 是否能解锁某家具，取决于：
  - 是否已点击到对应家具热区
  - 当前碎片数是否达到要求

## 7. 提示与引导的联动

当模块解锁时，系统会同步触发对应提示：

- 解锁生活区：`tab_living`
- 解锁户外区：`tab_forest`
- 解锁图鉴：`collection`
- 解锁娱乐模块：`arcade_entry`
- 首次进入娱乐模块后：`arcade_coin`
- 推币机通关后解锁抓娃娃机：`arcade_claw`

这些提示主要用于：

- 高亮当前应点击的入口
- 引导玩家按主线顺序体验

## 8. 老存档兼容说明

当前版本已补充娱乐模块老存档兼容逻辑。

如果旧存档已经有以下任一迹象：

- 主线已到 `arcade / claw / free`
- 已点过街机热点
- 已出现过 `arcade_entry / arcade_coin / arcade_claw` 相关提示
- 林海记录已达到娱乐模块应解锁的阶段

系统会在进入游戏时自动补回：

- `arcade`
- `coin`
- `claw`（如果存档已经推进到抓娃娃阶段）

这样可以避免“之前能玩，现在却显示未解锁”的误判。

## 9. 一句话总结

当前版本的推荐解锁路径为：

`办公区首次探索 -> 解锁生活区 -> 生活区首次休整/补给 -> 解锁户外区 -> 林海记录 3 个动物 -> 解锁娱乐模块 / 推币机 -> 完成 1 局推币机 -> 解锁抓娃娃机 -> 完成 1 局抓娃娃机 -> 进入自由培养`
